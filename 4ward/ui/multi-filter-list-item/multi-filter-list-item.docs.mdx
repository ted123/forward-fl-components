A component that does something special and renders text in a div.

### Component usage
```js
<MultiFilterListItem
  item={option}
  selected={values}
  onChange={(e) => handleOnSelectItemChange(e, option)}
>
  {option.label}
</MultiFilterListItem>
```

### Render Multi-Filter List Item!

```js live
<MultiFilterListItem item={[{key: '1',
  value: '1',
  label: 'The little girl'}]}>
  The little girl
</MultiFilterListItem>
```
