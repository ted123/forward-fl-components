import { InputDatepicker } from './input-datepicker';

Date picker component with option to input date via input field.
React-datepicker properties can also be used in this component, more info here:
[Docs](https://github.com/Hacker0x01/react-datepicker/blob/master/docs/index.md) & [Examples](https://reactdatepicker.com/)

### Component usage
```js
<InputDatepicker onChange={(date) => console.log(date)}/>
```

### Render InputDatepicker

```js live
<div style={{height: '300px'}}>
    <InputDatepicker onChange={(date) => console.log(date)}/>
</div>
```
